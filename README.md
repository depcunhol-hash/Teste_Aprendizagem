## Conceitos Fundamentais
- Controlo de versão (Git): Funciona como uma máquina do tempo ⏳ que tira "fotos" (commits) do seu trabalho, permitindo ver o histórico e voltar atrás se algo correr mal
- Fork (Bifurcação): É o ato de criar uma cópia de um projeto de outra pessoa para a sua própria conta, para que possa fazer experiências sem alterar o original
- Pull Request (Pedido de Alteração): É o processo de enviar as suas melhorias de volta para o dono do projeto original, pedindo-lhe que as aceite e inclua no trabalho principal
- Issues: Discussão de problemas conhecidos

## Ferramentas no Github

| **Ferramenta** | **O que faz?** | **Nível de dificuldade** |
| :---: | :---: | :---: |
|Editor Web|Apenas altera o texto|Muito Fácil|
|GitHub Pages|Mostra o seu site ao vivo|Fácil|
|Codespaces|Corre o código num computador virtual|Médio|
|Actions|Automatiza testes e tarefas|Avançado|
